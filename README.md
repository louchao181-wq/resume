# 楼超 AI 数字人简历 Demo

这是一个纯静态前端 demo，用于展示“AI 数字分身 + HR 简历问答 + 简历亮点卡片”的第一版效果。

## 打开方式

直接双击 `index.html`，或在浏览器中打开：

```text
C:\Users\12502\Documents\New project\ai-digital-human-demo\index.html
```

## 当前能力

- 左侧展示 3D 数字人、求职方向和高信号指标
- 中间支持 HR 快捷问题与自由输入问答
- 右侧支持项目、实习、教育、技能、游戏模块切换
- 问答内容已基于 `楼超_AIRB.pdf` 简历提炼
- 3D 数字人优先使用 Three.js 渲染，支持轻微呼吸、点头、鼠标视线跟随和回答时说话动效
- 如果 Three.js CDN 暂时不可用，会自动回退到本地 WebGL 版本，避免展示区域空白
- 游戏模块展示《三角洲行动》跑刀玩法拆解案精选页面，并提供完整拆解案与游戏产品简历下载
